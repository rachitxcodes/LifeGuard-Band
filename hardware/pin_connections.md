## ESP32 Pin Mapping

| Component     | ESP32 Pin |
|--------------|----------|
| MPU SDA      | 21       |
| MPU SCL      | 22       |
| GSM TX       | 17       |
| GSM RX       | 16       |
| GPS TX       | 26       |
| GPS RX       | 27       |
| SOS Button   | 13       |
| Buzzer       | 5        |
| Yellow LED   | 2        |
| Red LED      | 4        |

Power connections use 3.3V and GND appropriately.
